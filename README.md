# adamvanlente.com

Personal website.  Made using Vue, using the lovely [vuepack](https://github.com/egoist/vuepack) generator.

[adamvanlente.com](http://www.adamvanlente.com)

To start:

```bash
$ npm install
```

To develop:

```bash
$ npm run dev```

To build for production:

```bash
$ npm run build
```
