I love keeping up with Y Combinator's Hacker News feed. I thought it would be fun to create a simple web app to view the newest stories. Though there are a few apps out there for this, I wanted to test some nuances of media queries in CSS for responsive layouts. I spent some time making adjustments specific to iPhone, iPad, iPad Mini and a Nexus 7 tablet. Check it out and keep up to date with Hacker News at HackerNewsReader.com.

[UPDATE 2014]: I recently updated this app, and have made the code available on GitHub. Enjoy!
